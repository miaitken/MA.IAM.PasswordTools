# MA.IAM.PasswordTools

A PowerShell module designed to provide password related information in an Active Directory or Entra ID environment.


# Current Use Cases

Generate a psuedo-random password with Generate-MAPassword
Get the ages of AD user passwords with Get-MAPasswordAgeAD
Get the ages of Entra ID user passwords with Get-MAPasswordAgeEntra


# Installation

Download the MA.IAM.PasswordTools folder and all of its contents
In PowerShell...
    cd C:\Path\To\Downloaded\Folder
    Import-Module .\MyIAMModule.psd1 -Force
